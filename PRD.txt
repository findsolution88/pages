프로젝트명

루틴 안내 기기 추가 수요조사용 랜딩페이지 제작

1. 프로젝트 배경

창업자는 미취학 자녀 가정의 등원 및 취침 준비 과정에서 발생하는 반복 지시, 전환 어려움, 시간 지연 문제를 줄이기 위한 가정용 루틴 안내 기기를 준비 중이다.

기존 청년창업사관학교 지원 문서에는 공공데이터, 커뮤니티 사례, 인터뷰 기반 수요근거가 포함되어 있었으나, 예비창업패키지 및 창업중심대학 지원을 위해서는 추가 수요조사 결과를 확보할 필요가 있다.

이를 위해 짧은 설명형 랜딩페이지를 제작하고, 관심 있는 양육자가 최소한의 정보만 입력하여 추가 수요조사에 참여할 수 있도록 한다.

2. 프로젝트 목표
핵심 목표

등원/취침 준비 문제에 공감하는 부모가 페이지 내용을 쉽게 이해하도록 한다

제품의 사용 장면을 10초 이내에 이해할 수 있게 한다

방문자가 부담 없이 추가 수요조사에 응답하도록 유도한다

사업적 목표

추가 수요조사 응답자 확보

응답자 중 목표고객 비율 확인

필요성 공감 여부 확인

테스트 참여 의향 및 출시 안내 희망 여부 확인

가능하면 예상 구매 가능 가격대 확인

3. 비목표

이번 페이지에서 하지 않는 것:

정식 제품 판매

결제 기능

회원가입/로그인

관리자 페이지

고객 DB 구축

브랜드 소개용 다중 페이지 사이트

제품 상세 스펙 안내

과도한 퍼널 분석 기능

CTR, 세션 분석 중심 마케팅 페이지 구축

이번 프로젝트는 1페이지형 추가 수요조사 랜딩페이지로 제한한다.

4. 핵심 타깃
1차 타깃

4~7세 자녀를 둔 부모

등원 준비 시 반복 지시가 많은 가정

놀이에서 준비 행동으로 넘어갈 때 실랑이가 잦은 가정

말로 여러 번 알려줘야 하는 상황에 피로감을 느끼는 양육자

2차 타깃

취침 준비에서도 유사한 어려움을 겪는 가정

시각적 안내나 루틴표에 관심이 있는 부모

아이가 말보다 순서 안내에 더 반응하는 편이라고 느끼는 가정

제외 대상

단순 정보 소비자

문제 공감은 있으나 자녀 연령대가 너무 벗어나는 경우

육아 전반 정보만 탐색하는 비타깃 유입

5. 핵심 메시지
메인 메시지

아이의 등원·취침 준비를 말 대신 화면 순서로 안내하는 가정용 루틴 도구

문제 메시지

아침마다 같은 말을 여러 번 하게 된다

준비는 해야 하지만 다음 행동으로 잘 넘어가지 않는다

부모는 재촉에 지치고, 아이도 실랑이로 피로해진다

해결 메시지

집 안에 고정된 화면 순서 안내를 통해 아이가 다음 행동을 더 쉽게 인식하도록 돕는다

부모의 반복 지시 부담을 줄이고, 준비 과정을 더 예측 가능하게 만든다

현재 단계 메시지

현재는 정식 판매 전, 실제 필요한 가정이 있는지 확인하는 단계이다

간단한 추가 수요조사를 통해 초기 반응을 확인하고 있다

6. UX 원칙

첫 화면에서 무엇을 위한 페이지인지 바로 이해되어야 한다

기능 설명보다 사용 장면 중심으로 보여준다

“AI”, “플랫폼”, “혁신” 같은 표현보다 생활 문제를 먼저 보여준다

페이지는 짧고 단순해야 한다

방문자에게 많은 정보를 요구하지 않는다

광고 페이지보다 의견 요청 페이지처럼 보여야 한다

모바일에서 먼저 보기 쉽게 설계한다

7. 페이지 구조
섹션 1. 히어로

목적: 첫 화면에서 제품 개념과 문제 상황을 직관적으로 이해시키기

포함 요소:

메인 헤드라인

서브카피

간단한 태블릿/루틴 화면 이미지 또는 목업

메인 버튼

예시 헤드라인:

아이의 등원 준비를 말 대신 화면 순서로 안내하는 가정용 루틴 도구

반복 지시를 줄이고, 아이가 다음 행동을 더 쉽게 따라가도록 돕는 루틴 안내 기기

예시 버튼:

간단한 수요조사 참여하기

의견 남기기

섹션 2. 문제 공감

목적: 부모가 “우리 집 이야기 같다”는 느낌을 받게 하기

포함 요소:

짧은 문제 문장 3~4개

카드형 또는 리스트형 표현

예시 문장:

아침마다 같은 말을 여러 번 하게 됩니다

준비는 해야 하지만 다음 행동으로 넘어가기 어렵습니다

재촉이 반복되면서 부모와 아이 모두 지칩니다

말보다 눈에 보이는 순서가 더 잘 먹히는 순간이 있습니다

섹션 3. 해결 방식 소개

목적: 제품이 어떻게 쓰이는지 쉽게 설명하기

포함 요소:

집 안에 두고 쓰는 태블릿 형태 설명

아이가 화면 순서를 따라가는 구조 설명

부모와 아이 각각의 기대 효과

예시:

거실이나 현관에 두고 쓰는 안내형 태블릿 구성

아이는 화면에 보이는 순서를 따라 준비를 진행

부모는 반복해서 말로 지시하는 부담을 줄일 수 있음

섹션 4. 사용 예시

목적: 실제 사용 장면을 떠올릴 수 있게 하기

예시 시나리오:

등원 준비: 세수 → 옷 입기 → 가방 챙기기 → 신발 신기

취침 준비: 장난감 정리 → 씻기 → 책 읽기 → 잠자리 준비

표현 방식:

큰 아이콘 4개

짧은 단계 설명

체크 표시 느낌의 간단한 UI

섹션 5. 현재 모집 목적

목적: 정식 판매가 아니라 추가 수요조사 목적임을 명확히 전달하기

포함 요소:

현재 단계 설명

어떤 사람의 의견이 필요한지

왜 의견을 받고 있는지

예시 문장:

현재는 창업 준비 단계에서 실제 필요한 가정이 있는지 확인하고 있습니다

등원 또는 취침 준비가 특히 힘든 가정의 의견을 듣고 있습니다

아래 간단한 수요조사에 응답해주시면 제품 방향을 구체화하는 데 큰 도움이 됩니다

섹션 6. 수요조사 참여 유도

목적: 부담 없이 폼으로 이동시키기

요구사항:

페이지 중간 1회, 하단 1회 버튼 배치

버튼 클릭 시 외부 폼 링크 이동

버튼 문구는 가볍게 유지

예시 버튼 문구:

간단한 수요조사 참여하기

의견 남기기

출시 전 안내받기

섹션 7. 최소 신뢰 요소

목적: 과도한 상업 페이지처럼 보이지 않도록 최소한의 신뢰를 제공하기

포함 요소:

창업 준비 단계라는 설명

실제 부모 인터뷰와 수요 확인을 바탕으로 기획 중이라는 문구

응답 정보는 제품 검증 목적 외 사용하지 않는다는 안내

주의사항:

허위 후기 금지

과장된 성과 금지

허위 숫자 금지
